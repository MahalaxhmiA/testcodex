

## Scenario
Automated product search and validation on Amazon.in with strict business rules.

## Requirements Implemented
- **Time-Based Execution:** Only runs between 15:00 - 18:00.
- **Alphabet Filter:** Automatically skips products starting with A, B, C, or D.
- **Category Filter:** Excludes electronic items using keyword detection.
- **Validation:** Performs cross-tab verification of product titles.
